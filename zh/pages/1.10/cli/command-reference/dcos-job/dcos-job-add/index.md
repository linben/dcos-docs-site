# 描述

添加作业。

# 统计

```bash
dcos job add <job-file> [OPTION]
```

# 选项

None.

# 位置实参

| 姓名、速记              | 默认 | 描述            |
| ------------------ | -- | ------------- |
| `<job-file>` |    | JSON 格式的作业定义。 |

# 父命令

| ------------------------------------------------- | ----------------- |
