# 描述

列出所有任务。

# 统计

```bash
dcos marathon task list <app-id> [OPTION]
```

# 选项

| 姓名、速记    | 默认 | 描述           |
| -------- | -- | ------------ |
| `--json` |    | 打印JSON格式的数据。 |

# 位置实参

| 名字，简写            | 默认 | 描述      |
| ---------------- | -- | ------- |
| `<app-id>` |    | 应用程序ID。 |

# 父命令

| 命令                                                          | 描述                  |
| ----------------------------------------------------------- | ------------------- |
| [dcos marathon](/1.10/cli/command-reference/dcos-marathon/) | 将应用程序部署和管理到DC / OS。 |
