# 描述

Rename a cluster in the DC/OS CLI.

# 统计

```bash
dcos cluster rename <name> <new_name>
```

# 位置实参

| Name, shorthand    | Default | Description             |
| ------------------ | ------- | ----------------------- |
| `<name>`     |         | DC/OS cluster name.     |
| `<new_name>` |         | New DC/OS cluster name. |
