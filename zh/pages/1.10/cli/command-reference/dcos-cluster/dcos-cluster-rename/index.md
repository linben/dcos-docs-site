# 描述

在DC / OS CLI中重命名群集。

# 统计

```bash
dcos cluster rename <name> <new_name>
```

# 位置实参

| 名字，简写         | 默认 | 描述             |
| ------------- | -- | -------------- |
| `<名称>`  |    | DC / OS群集名称。   |
| `<新名称>` |    | 新的DC / OS群集名称。 |
