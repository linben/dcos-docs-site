# 描述

从DC/OS CLI中移除已配置的集群。

# 统计

```bash
dcos cluster remove <name>
```

# 位置实参

| 名字，简写          | 默认 | 描述           |
| -------------- | -- | ------------ |
| `<name>` |    | DC / OS群集名称。 |

# 父命令

| 命令                                                        | 描述           |
| --------------------------------------------------------- | ------------ |
| [dcos cluster](/1.10/cli/command-reference/dcos-cluster/) | 管理DC / OS群集。 |
